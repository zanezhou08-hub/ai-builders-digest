### Anthropic Engineering
Scaling Managed Agents: Decoupling the brain from the hands

Anthropic introduces Managed Agents, a hosted service that runs long-horizon agents through interfaces designed to outlast any particular implementation. The key innovation is decoupling the "brain" (Claude and its harness) from the "hands" (sandboxes and tools) and the "session" (the log of session events).

This architecture allows:
- Many brains and many hands (scalability)
- Each component to fail independently (resilience) 
- Security tokens to be kept separate from execution environments
- Context to be stored durably outside the context window

"We virtualized the components of an agent: a session (the append-only log of everything that happened), a harness (the loop that calls Claude and routes Claude's tool calls to the relevant infrastructure), and a sandbox (an execution environment where Claude can run code and edit files). This allows the implementation of each to be swapped without disturbing the others."

https://www.anthropic.com/engineering/managed-agents
